# 🧑‍🎨 Iconos estándar de introducción

Tenemos un conjunto de iconos estándar para las páginas que ponemos en todas las páginas de introducción:

---

En la barra de navegación, notarás algunos iconos. Aquí está su significado:

| Icono | Significado |
|------|-------------|
| 📘   | Material de lectura - Si estás siguiendo una sesión dirigida por un instructor, probablemente ya hayan cubierto esto. |
| 👐   | Contenido práctico - Prepárate para hacer algo de trabajo. Deberías seguir estos pasos. |
| 🦸   | Contenido avanzado - Este contenido no se cubre durante el laboratorio, pero si estás interesado en aprender más, puedes revisarlo. |
